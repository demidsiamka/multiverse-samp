# Multiverse SA-MP Launcher

Стильный и современный лаунчер для мультиплеера SA-MP (GTA: San Andreas), построенный на базе библиотеки CustomTkinter. Позволяет динамически загружать актуальный список серверов из сети, управлять никнеймами и быстро устанавливать необходимые ASI-компоненты.

## ВАЖНО
Запускать от имени адмиинстратора

## Особенности

* **Динамический список серверов:** Автоматическое получение актуальных серверов по удаленной ссылке (с проверкой интернет-соединения).
* **Кастомизация графического интерфейса:** Темная тема с неоновой индикацией для каждого выбранного сервера.
* **Удобное управление:** Сохранение последних использованных никнеймов и пути к игре в локальный конфигурационный файл.
* **Встроенный установщик Vorbis:** Быстрая интеграция необходимых библиотек (`vorbis.dll`, `vorbisFile.dll`, `vorbisHooked.dll`) для корректной работы ASI-плагинов и мультиплеера.
* **Автоматизация реестра:** Лаунчер самостоятельно записывает выбранный никнейм в реестр Windows перед запуском игры.

## Требования

Для запуска лаунчера необходим Python версии 3.7 или выше, а также установленная библиотека `customtkinter`.

pip install customtkinter
(bash/cmd)
